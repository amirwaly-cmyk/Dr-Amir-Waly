# Dr Amir Waly — Logo Pack

Consultant Specialist General Practitioner
Final identity: classic serif wordmark with a medical-blue accent.

## What's inside

/svg   — Scalable vector files (best for web, print & design software)
/png   — High-resolution transparent PNGs (3×) + avatar/favicon
/favicon — Web favicon PNGs (16, 32, 180, 512 px)
Brand Sheet.html — Self-contained overview of all variants, palette & clear-space (opens in any browser, offline)

## Logo variants

- primary    — full name + credentials, dark ink (use on light backgrounds)
- reversed   — white, for dark backgrounds
- on-blue    — white on the brand blue
- stacked    — compact version for tight spaces
- favicon-AW — "AW" monogram tile (app icon / browser tab)
- avatar-AW  — circular "AW" mark (social profiles)

## Colours

Ink            #23262B   — the name
Medical Blue   #4D72A6   — accent dot, favicon, highlights only
Blue Tint      #EAF0F7   — soft backgrounds
Cream          #F6F3EC   — warm neutral background

## Typography

Name        Spectral (Medium) — free Google Font
Credentials Work Sans (Medium) — free Google Font

Install both free fonts before editing the SVGs in design software, or ask
for an "outlined" version where the text is converted to paths (fully
self-contained, no font needed).

## Website favicon — quick setup

Place the favicon PNGs in your site root and add to <head>:

  <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
  <link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png">
  <link rel="apple-touch-icon" sizes="180x180" href="/favicon-180.png">

## Clear space

Keep clear space around the logo equal to at least the cap-height of the
name on every side. Don't recolour, stretch, or rotate the marks.
